# CSS 颜色

---

颜色是通过对红、绿和蓝光的组合来显示的。

---

### 颜色值

CSS 颜色使用组合了红绿蓝颜色值 (RGB) 的十六进制 (hex) 表示法进行定义。对光源进行设置的最低值可以是 0（十六进制 00）。最高值是 255（十六进制 FF）。


十六进制值使用三个双位数来编写，并以 # 符号开头。

### 1600 万种不同的颜色
