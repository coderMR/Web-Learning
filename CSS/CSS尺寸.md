### CSS 尺寸属性

CSS 尺寸属性允许你控制元素的高度和宽度。同样，还允许你增加行间距。

| 属性 | 描述
|------|-----
| height | 设置元素高度
| line-height | 设置行高
| max-height | 设置元素最大的高度
| max-width | 设置元素的最大宽度
| min-width | 设置元素的最小宽度
| min-height | 设置元素的最小高度
| width | 设置元素的宽度
